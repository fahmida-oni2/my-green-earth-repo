2) What is the difference between map(), forEach(), and filter()?
Answer: forEach() : Executes a function on each array element which does not return a new array.
        map() : Transforms each element and returns a new array of the same length.
        filter() : Check a condition and returns a new array with elements that pass a condition.

3) What are arrow functions in ES6?
Answer: Arrow functions are a concise ES6 syntax for writing functions with implicit return for single expressions. They support implicit return when the function has only one expression and make the code more cleaner.

4) How does destructuring assignment work in ES6?
Answer: Destructuring in ES6 allows to extract values from arrays or objects into variables in a concise way.It also provides a cleaner syntax.It helps to work with structured data .
For arrays, elements can be assigned directly.
For objects, properties can be extracted by name
